# CodeAlpha Chatbot Project

This is a basic chatbot developed using Python as part of the CodeAlpha Internship.

##  Features
- Responds to user messages
- Handles greetings and simple questions
- Continuous conversation using loop

## Concepts Used
- Python loops
- Conditional statements (if-elif)
- User input handling

## How to Run
1. Open in Python IDLE
2. Run the file
3. Start chatting
